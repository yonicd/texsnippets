# texsnippets

Add `RStudio` snippets for boilerplate TeX commands

All snippets have a ts (texsnippets) prefix. Write `'ts'` in the `RStudio` source editor and press `tab` button.

```
ts_figure
ts_table
ts_table
ts_tabular
ts_center
ts_array
ts_caption
ts_label
```

Where it is relevant the template below is used.

```
\begin{}

\end{}
```
